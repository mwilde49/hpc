# Test Fixtures

This directory contains scripts that generate or download test data for the
TJP HPC pipeline test suite (`tjp-test-suite`). Each fixture script is
designed to be run **once** before executing Layer 3 (full SLURM execution)
tests for the corresponding pipelines.

Fixtures do not ship with the repository because the generated data is either
too large (genomic FASTQs, reference indexes) or cannot be committed (synthetic
binaries). Run the scripts below on Juno to populate the data.

---

## Fixture scripts

| Script | Pipelines | What it produces | When to run |
|--------|-----------|-----------------|-------------|
| `generate_rnaseq_synthetic.sh` | `bulkrnaseq`, `psoma` | Synthetic paired FASTQ files (50 bp random reads) + `samples.txt` | Before L3 bulkrnaseq / psoma tests |
| `generate_virome_synthetic.sh` | `virome` | Synthetic paired FASTQ files + `samplesheet.csv` | Before L3 virome tests |
| `download_10x_fixtures.sh` | `cellranger-mkfastq` | 10x Genomics tiny BCL run folder + SampleSheet CSV | Before L3 cellranger-mkfastq tests |

## Fixture details

### generate_rnaseq_synthetic.sh

**Produces:**

```
$SCRATCH_ROOT/pipelines/rnaseq_synthetic/fastq/
  Sample_A_1.fastq.gz    R1 for Sample_A (1000 × 50bp random reads)
  Sample_A_2.fastq.gz    R2 for Sample_A
  Sample_B_1.fastq.gz    R1 for Sample_B
  Sample_B_2.fastq.gz    R2 for Sample_B
$SCRATCH_ROOT/pipelines/rnaseq_synthetic/samples.txt   (Sample_A, Sample_B)
```

**Usage:**

```bash
# Default output location (uses $SCRATCH_ROOT from environment)
source /groups/tprice/pipelines/bin/lib/common.sh
/groups/tprice/pipelines/test_data/fixtures/generate_rnaseq_synthetic.sh

# Custom output location with 500 reads per sample
/groups/tprice/pipelines/test_data/fixtures/generate_rnaseq_synthetic.sh \
    /scratch/juno/$USER/my_test_fastqs 500
```

**Important:** The reads are random ACGT and will not align to any real genome.
A matching STAR index (for bulkrnaseq) or HISAT2 index (for psoma) against a
tiny reference (e.g., chrM only) is required for a complete Layer 3 run.

Build a tiny STAR index:

```bash
STAR --runMode genomeGenerate \
     --genomeDir $REPO_ROOT/references/tiny/star_index \
     --genomeFastaFiles /path/to/chrM.fa \
     --sjdbGTFfile /path/to/chrM.gtf \
     --genomeSAindexNbases 7 \
     --runThreadN 4
```

Build a tiny HISAT2 index:

```bash
hisat2-build /path/to/chrM.fa \
    $REPO_ROOT/references/tiny/hisat2_index/tiny
```

### generate_virome_synthetic.sh

**Produces:**

```
$SCRATCH_ROOT/pipelines/virome/test_data/
  fastq/
    virome_sampleA_1.fastq.gz
    virome_sampleA_2.fastq.gz
    virome_sampleB_1.fastq.gz
    virome_sampleB_2.fastq.gz
  samplesheet.csv      (sample,fastq_r1,fastq_r2 — absolute paths)
```

**Usage:**

```bash
source /groups/tprice/pipelines/bin/lib/common.sh
/groups/tprice/pipelines/test_data/fixtures/generate_virome_synthetic.sh

# Custom location
/groups/tprice/pipelines/test_data/fixtures/generate_virome_synthetic.sh \
    /scratch/juno/$USER/virome_test 500
```

**Kraken2 database:** A Kraken2 database is required for a Layer 3 virome run
but is NOT generated by this script (databases are 5+ GB). Options:

- **Shared DB** (recommended): `/groups/tprice/pipelines/references/kraken2_viral_db`
- **Build tiny DB** (for isolated testing):

```bash
kraken2-build --download-taxonomy --db /scratch/juno/$USER/tiny_kraken2_db
kraken2-build --add-to-library viral_genome.fa --db /scratch/juno/$USER/tiny_kraken2_db
kraken2-build --build --db /scratch/juno/$USER/tiny_kraken2_db --threads 4
```

### download_10x_fixtures.sh

**Downloads:**

| File | Source |
|------|--------|
| `cellranger-tiny-bcl-1.2.0.tar.gz` | 10x Genomics public S3 |
| `cellranger-tiny-bcl-simple-1.2.0.csv` | 10x Genomics public S3 |

**Extracts to:**

```
$REPO_ROOT/test_data/10x/cellranger-mkfastq/
  tiny-bcl/                               Illumina BCL run folder
  cellranger-tiny-bcl-simple-1.2.0.csv   Simple SampleSheet CSV
```

**Usage:**

```bash
/groups/tprice/pipelines/test_data/fixtures/download_10x_fixtures.sh

# Custom destination
/groups/tprice/pipelines/test_data/fixtures/download_10x_fixtures.sh \
    --dest /scratch/juno/$USER/10x_fixtures

# Force re-download
/groups/tprice/pipelines/test_data/fixtures/download_10x_fixtures.sh --force
```

**Requirements:** `wget` or `curl` must be available. On Juno, both are
available as system utilities.

## What is NOT generated automatically

| Item | Reason | How to obtain |
|------|--------|---------------|
| STAR index (tiny) | Requires STAR binary + FASTA/GTF | `STAR --runMode genomeGenerate` (see above) |
| HISAT2 index (tiny) | Requires hisat2-build + FASTA | `hisat2-build chrM.fa <prefix>` |
| Kraken2 DB (tiny) | Large database | `kraken2-build` or use shared DB |
| Cell Ranger SIF | Not applicable (native tool) | Install from 10x tarball |
| wf-transcriptomes ONT FASTQs | Real sequencing data | Stage manually to `containers/sqanti3/test_data/wf_transcriptomes/fastq/` |
| Xenium output bundle | No minimal public dataset | Use real Xenium run output |
