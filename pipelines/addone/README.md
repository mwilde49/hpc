# AddOne Demo Pipeline

## Description
Reads a text file of numbers, adds 1 to each value, and writes output.

## Usage (Direct Python)
python addone.py --input input.txt --output output.txt

## Usage (Config Driven)
python addone.py --config example_config.yaml

## Purpose
Demonstrates:
- Apptainer container execution
- SLURM scheduling
- Config-driven pipelines
- Group HPC architecture
